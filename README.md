# Broadmead Site (broadmead-web)

Modern, conversion-first website rebuild for Broadmead Boutique B&B (Tenby).

## Stack
- Next.js (App Router) + TypeScript
- Tailwind CSS

## Getting started (local)
1) Install dependencies:
   npm install

2) Run dev server:
   npm run dev

3) Open:
   http://localhost:3000
